# Screenshot Guide

Take these screenshots for your MCA report and viva slides.

1. Login page
Show the professional landing UI and admin credentials card.

2. Student registration page
Capture the webcam registration panel and face capture step.

3. Admin dashboard
Show analytics cards and the exam creation form.

4. Published exam card in student dashboard
Show that the student can launch the exam.

5. Secure exam room
Include countdown, webcam monitor, and alert feed.

6. Suspicious activity evidence
Trigger a tab switch or no-face alert and capture the admin session details view.

7. PDF report open in browser
Show the generated per-student report.
